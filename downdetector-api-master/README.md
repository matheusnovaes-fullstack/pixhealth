✅ DNS Lookup (resolver o domínio para IP) - ~10-50ms

✅ Conexão TCP (estabelecer conexão) - ~20-100ms

✅ TLS/SSL Handshake (HTTPS) - ~50-200ms

✅ Envio da requisição (seu PC → servidor do banco atráves de axios) - ~10-50ms

✅ Processamento no servidor (banco processa) - ~50-500ms

✅ Resposta (servidor do banco → seu PC) - ~10-50ms

✅ Recebimento completo (download do HTML) - ~10-100ms

Total: ~160ms a ~1050ms (dependendo de cada etapa)
